# Flower Wrapping Paper Showcase

เว็บโชว์สินค้าแนว luxury สำหรับผลิตภัณฑ์กระดาษที่แปรรูปจากดอกไม้และเศษวัสดุ

## Products

1. Flower wrapping paper
2. Shoe Packaging Paper
3. Packaging

## Features

- Responsive layout (mobile / tablet / desktop)
- Product carousel เลื่อนอัตโนมัติทีละรายการ
- โทนสีขาว น้ำตาลอ่อน ทอง และชมพูอ่อน

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
